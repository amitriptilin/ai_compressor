# AI Academic Compressor

Сжимает и упрощает академические тексты из PDF с помощью нейросети DeepSeek (через OpenRouter).

## Установка

1. Установи Python 3.9+.
2. Склонируй репозиторий и перейди в папку.
3. Установи зависимости:
   pip install -r requirements.txt

## Использование

Графический интерфейс

   streamlit run app.py

Введи API-ключ, выбери PDF, настройки и запусти.
